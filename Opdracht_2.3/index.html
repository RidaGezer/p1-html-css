Opdracht 2.3: Onderzoeken

Waarom noem je de eerste HTML-pagina; index.html?

De eerste pagina heet index.html zodat de programma waarmee je codeert de website kan herkennen.

Met welke HTML versie werken we nu?

Wij werken nu met versie HTML 5.

Met welke CSS versie werken we nu?

Wij werken met CSS versie 3.

Hoe kan het dan een website er net iets anders uit ziet in een andere browser? (Edge, Chrome, Firefox, Opera)

Niet alle browsers interpreteren alle codes hetzelfde, waardoor een site er over het algemeen hetzelfde uit kan zien maar dat bepaalde details van een site van elkaar kunnen verschillen.

Welke browser is het populairst en hoeveel procent RAM gebruikt deze browser?

Google chrome is het meest populair op dit moment met een RAM verbuik van ongeveer 1.8 GB.
